# Evidence-Driven Delivery

A visual essay on evidence-driven delivery — replacing calendar-driven commitment with evidence-gated promotion.

## Local development

```bash
python3 -m http.server 8080 --bind 127.0.0.1
```

Then open [http://127.0.0.1:8080](http://127.0.0.1:8080).

## Structure

Single self-contained HTML file (`index.html`) with inline CSS and JavaScript. No build step required.
